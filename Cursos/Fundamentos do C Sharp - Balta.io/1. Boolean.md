- Armazena somente **true** ou **false**, verdadeiro ou falso
- Usado pela palavra reservada bool
- **bool** => 8 bits

**Exemplo:**
```C#
const bool userIsRegister = true;  
if (userIsRegister)  
{  
    Console.WriteLine("User is Register");  
}  
else  
{  
    Console.WriteLine("User is not Register");
}
```
